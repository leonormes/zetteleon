# Tech Stack

## Context

Tech stack for chezmoi dotfiles and developer environment configuration management.

- Configuration Manager: chezmoi 2.x+
- Shell: zsh (with oh-my-zsh/starship)
- Primary Language: Shell Scripts (bash/zsh)
- Configuration Languages: TOML, JSON, Lua, HCL
- Terminal: Wezterm with custom configuration
- Editor Integration: Neovim, VS Code, Cursor, Windsurf
- Package Manager: Homebrew (macOS), npm (Node tools)
- Node Version: Latest LTS (for MCP servers)
- MCP Framework: Model Context Protocol servers
- AI Integration: Claude Code, Cursor, Windsurf, Gemini
- Knowledge Management: Obsidian with MCP tools
- Memory System: Pieces.app for code snippets and context
- Git Management: GitKraken, command line
- Automation Tools: Hammerspoon (macOS automation)
- Infrastructure: Terraform, Kubernetes, Helm
- Container Platform: Docker Desktop
- Cloud Platforms: Azure, AWS
- CI/CD Platform: GitHub Actions, GitLab CI
- Documentation: Markdown with Obsidian
- Configuration Hosting: Git repositories
- Deployment: chezmoi apply for dotfiles sync
- Environment: macOS primary, Linux secondary
