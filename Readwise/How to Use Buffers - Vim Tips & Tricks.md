## Highlights
- **Buffer Management Instructions in Vim**
  - **Switch to the next buffer:** Use the command `:bnext` or its shortcut `:bn`. 
  - **Switch to the previous buffer:** Use `:bprevious` or `:bp`. 
  - **Jump to a buffer by index:** Use `:b` followed by the index number, e.g., `:b 1`. 
  - **Switch to a buffer by name or partial name:** Use `:b` followed by part of the buffer's name, e.g., `:b ori`. 
  - **Jump to the first or last buffer:** Use `:bfirst` or `:blast`. 
  - **Close a buffer without closing Vim:** Use `:bd` followed by the buffer index or name, e.g., `:bd 1`. 
  - **Create a new buffer (unsaved):** Use `:e` followed by a filename or `:enew` for a new unnamed buffer. 
  - **Set 'hidden' option to allow buffer switching without saving:** Use `:set hidden`. 
  - **Use buffer index for quick navigation:** Use `b` command with index, e.g., `:b 3`.
