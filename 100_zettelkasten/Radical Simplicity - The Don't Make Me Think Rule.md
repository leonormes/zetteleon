## Minimal Components

The system should have very few moving parts. Instead of multiple apps, tags, complex folder structures, and intricate linking, it focuses on one or two core tools or methods that are incredibly straightforward.

## Intuitive Workflow

The process of capturing, organising (if at all), and retrieving information should be so obvious that it requires almost no conscious thought or recall of specific rules. If you have to stop and remember "How do I file this?" or "What tag should I use?", the friction is already too high.
