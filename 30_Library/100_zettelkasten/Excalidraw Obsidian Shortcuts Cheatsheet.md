## Essential Keyboard Shortcuts for Excalidraw in Obsidian

| Action | Shortcut |
| :-- | :-- |
| Select | V |
| Pen/Freehand | P |
| Eraser | E |
| Rectangle | R |
| Ellipse | O |
| Line | L |
| Arrow/Connector | X |
| Text Tool | T |
| Image Import | I |
| Duplicate | Ctrl/Cmd + D |
| Undo | Ctrl/Cmd + Z |
| Redo | Ctrl/Cmd + Shift + Z |
| Group/Ungroup | Ctrl/Cmd + G / Shift + Cmd + G |
| Bring Forward/Backward | Ctrl/Cmd + ] / Ctrl/Cmd + [ |
| Copy/Paste | Ctrl/Cmd + C / Ctrl/Cmd + V |
| Save Drawing | Ctrl/Cmd + S |
| Edit text box | Enter |
| Confirm edit/Insert link | Ctrl/Cmd + Enter / Cmd + K |
| Delete selected | Delete/Backspace |

## Obsidian-Specific Features

- **Insert Obsidian Note Link**: Cmd/Ctrl + K
- **Create Markdown Note from Shape**: Right-click on shape → “Convert to file”
- **Zoom In/Out**: Ctrl/Cmd + Wheel / Ctrl/Cmd + +/-
- **Export to PNG/SVG**: Cmd/Ctrl + Shift + E
